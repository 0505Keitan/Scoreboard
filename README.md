# Scoreboard

![CI](https://github.com/0505Keitan/Scoreboard/workflows/Node%20CI/badge.svg)

## Requirements
 - Node.js 10.x, 11.x
 - express
 - http
 - socket.io

## Installation
1. Clone this repository.
2. Run `npm install`.
3. Run `node index.js`.

## Notes
This project's CI is only run `npm install`.
