シンプルなスコアボード

さまざまな場面で利用できるスコアボードです。2つ以上デバイスを用意するだけで簡単に利用ができます。

使いやすい操作画面

ボタンは最低限にし、どのサイズのデバイスでも押しやすいように大きめに設計。

見やすいスコア表示画面

プレイヤー名とスコアだけのシンプルな構成。

面倒な会員登録必要なし！

面倒な会員登録は必要ありません。デバイスを２つ以上用意すればすぐに利用できます！

つかいかた

  1. 左上の「コントロールパネル」か ここ からコントロールパネルにアクセスします。
  2. 結果表示画面にアクセスするための あいことば と プレイヤー名 を設定してURLを発行します。あいことばを指定しない場合は自動生成されます。また5文字以下にはできません。
  3. 表示用URLに結果表示に使いたいデバイスでアクセスします。
  4. あとはスコアに応じてボタンを押していくだけ！リセットを押すとスコアがリセットされます。

使用上の注意

  • あいことば や プレイヤー名 にはパスワードや個人を特定できるものなどを利用しないでください。
  • バグ等がありましたら、 GitHub にお願いします。
  • 利用中に何らかの損害が生じた場合も運営者は責任を負いません。
